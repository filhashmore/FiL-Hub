## Audio Tools

### RF Scout
- **URL:** https://rf-scout.vercel.app
- **Status:** Beta
- **Stack:** React, Vite, Tailwind, Framer Motion, FCC API
- RF frequency coordination tool for touring audio professionals. Real-time spectrum visualization with FCC API integration, automatic IM product avoidance, TV channel exclusion, and export to Shure WWB & Sennheiser WSM formats.
- Frequency Range: 470–608 MHz | 10+ Preset Venues

### SetFlow
- **URL:** https://setflow-beryl.vercel.app
- **Status:** Beta
- **Stack:** React, Vite, TypeScript, Tailwind, shadcn/ui
- Setlist builder with flow analysis and PDF export. Build setlists with BPM/key metadata, Circle of Fifths key clash detection, energy analysis, and stage-optimized PDF export for low-light environments.

## SaaS

### Filmore Advance Portal
- **URL:** https://filmore-advance.vercel.app
- **Status:** Beta
- **Stack:** React, Vite, Tailwind, Supabase
- **Access Codes:** DEMO, FLY1, TRIO
- Self-service venue advance portal for production logistics. Venues access tech riders, stage plots, hospitality riders, and production docs with profile-based content filtering for different show types.

### Tour Advance Portal
- **URL:** https://tour-advance-portal.vercel.app
- **Status:** In Development
- **Stack:** React, Vite, TypeScript, Supabase
- Reusable advance portal template for touring artists. Config-driven, multi-tenant template enabling any production manager to deploy their own venue advance system.

### Filmore Gear Inventory
- **URL:** https://filmore-gear.vercel.app
- **Status:** Beta
- **Stack:** React, HTML/CSS, localStorage, Tailwind
- **Access Code:** DEMO4
- Comprehensive gear tracking for touring production. Real-time inventory tracking with consumables management, fly pack assignments, and role-based acquisition approvals.

### Tour Flow App
- **Status:** Paused (no live URL)
- **Stack:** React Native, Expo, TypeScript, Supabase, Zustand
- Mobile tour management for audio engineers and crew. Multi-crew collaboration, tour scheduling, gear inventory, input list management, and AI assistant integration.
